# Yacut

Yacut — это укоротитель ссылок с поддержкой загрузки файлов на Яндекс.Диск и генерацией коротких URL для файлов и обычных ссылок.

## Возможности

* Генерация коротких ссылок для любых URL
* Загрузка файлов на Яндекс.Диск с генерацией коротких ссылок для скачивания

## Технологии

* Python 3
* Flask
* SQLite
* Asyncio
* Яндекс.Диск API

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/eqsx000111/async-yacut-main
cd yacut
```

### 2. Создать и активировать виртуальное окружение

```bash
python3 -m venv venv
```

Для Linux/macOS:

```bash
source venv/bin/activate
```

Для Windows:

```bash
venv\Scripts\activate
```

### 3. Установить зависимости

```bash
python3 -m pip install --upgrade pip
pip install -r requirements.txt
```

### 4. Создать файл `.env` в корне проекта

В файле `.env` должны быть следующие переменные окружения:

```env
FLASK_APP=yacut
FLASK_ENV=development
SECRET_KEY=your_secret_key
DB=sqlite:///db.sqlite3
DISK_TOKEN=ваш_токен_Яндекс_Диска
```

**Важно:** Не забудьте указать токен Яндекс.Диска для работы с API загрузки файлов.

Для получения токена Яндекс.Диска:
1. Перейдите на https://oauth.yandex.ru/
2. Зарегистрируйте приложение и получите OAuth-токен

### 5. Создать базу данных и применить миграции

```bash
flask db upgrade
```

### 6. Запустить проект

```bash
flask run
```

После этого проект будет доступен по адресу: http://127.0.0.1:5000


## Использование

### Создание короткой ссылки

1. Откройте главную страницу
2. Введите оригинальную ссылку
3. (Опционально) Укажите кастомный короткий идентификатор
4. Нажмите "Создать"

### Загрузка файла на Яндекс.Диск

1. Перейдите на страницу загрузки файлов
2. Выберите файл(-ы) для загрузки
3. Файл(-ы) будет автоматически загружен на Яндекс.Диск
4. Получите короткую ссылку для скачивания
